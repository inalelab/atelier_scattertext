# Atelier Scattertext pour la comparaison de corpus d'émissions TV

Support pour un atelier de découverte du package Python [Scattertext]((https://github.com/JasonKessler/scattertext), et un exemple d'application à l'analyse comparative du discours des chaînes TV d'information en continu françaises.
Scattertext propose une visualisation très simple et intuitive des termes sous forme de nuages de points répartis en fonction de leur fréquence dans deux corpus A et B. Il introduit également une mesure d'association spécifique de termes à une variable. Cette bibliothèque peut être utilisée pour des analyses exploratoire lorsqu'il s'agit de comparer deux corpus (avant/après, chaîne A/chaîne B, deux locuteurs) ou les résultats de classifications (analyse de sentiment...).

Le code dans ce dépôt s'appuie sur un jeu de données au format .CSV, non fourni, qui porte sur 1 mois x 5 chaînes d'info en continu françaises. L'accès à ces données est soumis à la signature d'une convention de recherche avec [le lab INA](https://inalelab.hypotheses.org/).
